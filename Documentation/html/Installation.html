Installation¶

Automatic setup¶

To perform the standard Setup.exe, please launch the Setup.exe file in the main directory. You can choose the modules to install checking the corrresponding box. This programm works with a Python3.4 or greater distribution. Follow the instructions and wait until the setup is finished. It may take several minutes, please be patient.

Manual setup¶

You can find in each module a setup directory containing the local dependencies of the concerned module. Launch the setup.bat script and wait until the installation is finished. Once done you can load the specific module you have installed by using the module_name.exe file (recommanded), using the launch it via the module_name.bat script file OR launch via a python console (advaned users).
