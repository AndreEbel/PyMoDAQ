Data Viewers
============

.. currentmodule:: pymodaq.daq_utils.plotting.data_viewers

The data viewers are classes devoted to data display from scalar data up to 4 dimensions data. All the viewers inherits
from the base class `ViewerBase` and then offers options and interactions depending their dimensionality
