Installation
------------

This program need the python 3.4 or greater distribution.
You could find such at https://www.python.org/
To perform global install, launch the setup.exe file in the main direcory.

To install with command prompt, in each module directory you will find 
a setup floder.
run the 'python setup.py install' command to setup the local module
and build external dependencies.
To add the module to your python library :
launch the 

<module_name>-1.0.win-amd64.exe file

or

PyMoDAQ_dependencies-1.0.win-amd64.exe 

to add all modules to your python library.

It will add the module to your python site-package library folder,
accessible from any python console.

Quick-Start
-----------
Once installed with setup you could launch any module from the desktop shortcuts or
python command using "python <module_name.py>.
In case of command prompt install launch via the local .exe file or .bat file.
