DAQ_Analysis
============

Introduction
------------


| **The DAQ_Analysis module is a .h5 file pre-viewer.**
| 
| H5 files are the storing data of experiment data results from the DAQ_Scan module.
| h5 files are structured in a tree, each file contain one or differents scan raw values, and a logging general informations.
| Each scan contain a save of the move value and datas corresponding.
| Data are viewable into the differents instance of viewers (respectively 0D,1D and 2D).
| The program is composed of one main file and work independantly of the others DAQ modules.
|
