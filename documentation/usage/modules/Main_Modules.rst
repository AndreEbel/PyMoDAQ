The 5 modules below are the main modules used in PyMoDAQ. DAQ_Move and DAQ_Viewer can be used as stand alone user
interface to manually control hardware. DAQ_Viewer can be used like this to monitor and/or log data from a specific
detector. However, the main use is through the DashBoard module and its functionalities given by the DAQ_Scan module
that will be used to perform automatic data acquisition and by the DAQ_Logger module to perform advanced data logging.
The usage of these modules is explained below.
